📂
Clicca per scegliere oppure trascina il file CSV qui
Formati supportati: .csv
Incolla l'URL del foglio Google Sheets. Il foglio deve essere condiviso ("chiunque abbia il link può visualizzare") oppure pubblicato sul web (File → Condividi → Pubblica sul web).
Il dashboard converte automaticamente qualsiasi URL Sheets nel formato CSV corretto.
Aggiornamento automatico:
Nota: Se il foglio è privato (condiviso solo con account specifici) la connessione diretta non funziona — usa il tab Carica file CSV. Per connessioni sicure a fogli privati consulta la guida nel tab Come funziona.
Apri Google Sheets → File → Scarica → Valori separati da virgola (.csv) → apri il file con un editor di testo → copia tutto → incolla qui.
Opzione A — Connessione diretta (URL)
1
Apri il foglio Google Sheets
2
Clicca File → Condividi → Condividi con altri → imposta "Chiunque abbia il link" → Visualizzatore
3
Copia l'URL dalla barra del browser
4
Incollalo nel tab 🔗 Google Sheets e clicca Connetti
5
Imposta l'aggiornamento automatico — il badge in alto indica l'ultimo sync
✓ Funziona da Cloudflare Pages, Vercel, GitHub Pages, qualsiasi hosting pubblico.
✗ Non funziona da file locale (file://) per limiti CORS del browser.
Opzione B — Carica file CSV (sempre disponibile)
1
Apri Google Sheets → foglio RACCOGLITORE
2
Clicca File → Scarica → CSV
3
Nel tab ↑ Carica file CSV trascina o seleziona il file
✓ Funziona sempre, anche offline e da file locale.
⚠ I valori Tally con "•" vengono rimossi automaticamente.
